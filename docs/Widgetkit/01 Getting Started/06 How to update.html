How to update

Updating Widgetkit is pretty straightforward. Just follow these steps:

For Joomla

Just download the latest Widgetkit release and use the Extension Installer like already described in {doc: install-widgetkit text:Install Widgetkit}.

For Wordpress

To update Widgetkit just follow these steps

  1. Download the latest Widgetkit version and unpack files and folders in the archive in a temporary directory on your local hard drive called widgetkit.
  2. Using FTP, upload the widgetkit folder (including files and subdirectories) to the directory \wp-content\plugins\ on your server, just override the old folder.
