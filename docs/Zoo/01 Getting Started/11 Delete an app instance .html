Delete an app instance

This is a short tutorial on how to delete an app instance in your ZOO extension.

Go to the ZOO administration and click on the gearwheel to open the app manager.

Select the app you have an instance of, which you want to delete.

Click on the button Info and you will see all created instances of this app.

Check the app instance you want to delete and click on Delete in the top right corner.

Please note, that all data in this app instance, e.g. items, categories etc. will be deleted too.
