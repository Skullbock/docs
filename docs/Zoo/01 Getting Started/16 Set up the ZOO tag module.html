Set up the ZOO tag module

Learn how to set up and configure the ZOO tag module for your site. You can show tags of your app in a Joomla module and publish it on a module position in your template. You can choose between different styles and settings.

When you install the ZOO extension, the ZOO tag module is installed automatically.

To create a new ZOO tag module, log in to the Joomla administration and click on Extensions » Module Manager. Click on ZOO Tag

On the left, you have the typical configurations of a Joomla module. Give the module a name, enable it and choose the module position for it.

On the right you can set the module parameters. Here is a description of the fields.

  1. Theme: Select how the items should look like in the ZOO tag module. Choose between the predefined themes or create your own one. To do so, check out {doc: create-a-new-layout-with-positions text: this tutorial}.
  2. Application/Category: Here you can choose the app and one of its categories of which you want to show tags in this module. In the next option you can set, whether the tags of the subcategory of the chosen one should be included or not.
  3. Tag Count: Choose, how many tags you wan to show in the module.
  4. Tag Order: In this option you determine the order of the displayed tags.
  5. Module Class Suffix: Set template specific module class suffixes to change the appearing of the module in the frontend.

Click on Save in the top right corner and the tag module will be shown to your visitors.
