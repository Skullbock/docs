Create a new module layout

This tutorial will take a closer look at the module layouts, what they are and how to create a new one.

Module Layouts

Module Layouts define the alignment and proportions of a set of modules in a specific position. For example they can be vertical or horizontal aligned.

Location

You can find the default module layouts provided by Warp in following folder: THEME_NAME/layouts/modules/layouts/

Default Layouts

By default Warp offers 3 different module layouts.

equal.php

Every module has the same dimension and they are displayed horizontal, next to each other. We have added support for up to 6 modules. You can add more of course.

Example, 4 modules with the same size (100 divided by 4 = 25)

case 4:
  printf('<div class="grid-box width25 grid-h">%s</div>', $modules[0]);
  printf('<div class="grid-box width25 grid-h">%s</div>', $modules[1]);
  printf('<div class="grid-box width25 grid-h">%s</div>', $modules[2]);
  printf('<div class="grid-box width25 grid-h">%s</div>', $modules[3]);
  break;

stack.php

All modules have a full width and will displayed vertical, one below the other.

Example:

  printf('<div class="grid-box width100 grid-v">%s</div>', $module);

double.php

The double layout has different module proportions. For example when you have 3 modules, the first one has double the size than the other two.

Example: 3 modules, first one has double size

case 3:
  printf('<div class="grid-box width50 grid-h">%s</div>', $modules[0]);
  printf('<div class="grid-box width25 grid-h">%s</div>', $modules[1]);
  printf('<div class="grid-box width25 grid-h">%s</div>', $modules[2]);
  break;

How to set a module layouts in a special position

In our Warp Administration Interface, you are able to give module positions different module layouts. Just click on "Profiles", scroll down to the "Layout Setting" and select the module layout you need for the different positions.

How to add a new module layout?

To create a new module layout add a new file to THEME_NAME/layouts/modules/layouts/ or if you are using a theme style to THEME_NAME/styles/YOUR_STYLE/layouts/modules/layouts/

Best way is to copy an existing module layout and modify it. For example copy THEME_NAME/warp/layouts/modules/layouts/double.php to THEME_NAME/layouts/modules/layouts/double.php . This new and modified module layout will override the default one.

Last but not least if you created a new module layout you have to add it to the theme parameters so it can be selected in the theme administration. To do so open config.xml and look for following code:

  <field name="top-a" type="list" default="equal" label="Top A Layout" description="Select a grid layout for this module position.">
     <option value="equal">Equal</option>
     <option value="stack">Stack</option>
     <option value="double">Double</option>
     <option value="custom01">Custom 01</option>
  </field>

  <field name="top-b" type="list" default="equal" label="Top B Layout" description="Select a grid layout for this module position.">
     <option value="equal">Equal</option>
     <option value="stack">Stack</option>
     <option value="double">Double</option>
     <option value="custom01">Custom 01</option>
  </field>
