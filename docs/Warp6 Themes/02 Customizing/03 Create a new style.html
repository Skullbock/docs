Create a new style

In this tutorial we show you how to create a new style for your Warp theme.

Creating a new style

Let's create a new style we call fresh. To do so we have to create a new style folder THEME_NAME/styles/fresh/. That's it! We created a new style which looks the same like our default theme.

Next you can add your own CSS /css/layout.css file, by copying it to THEME_NAME/styles/fresh/css/layout.css. The "fresh" style would now load it's own layout style sheet. But what if you only have one small modification? For example you just want to change one color. Do you have to copy the whole layout CSS from the default theme? No, just import the /css/layout.css by using @import url(../../../css/layout.css); and add your small modification. You have full control which CSS assets you want to include from the default theme, from the core framework or neither.

We can override the default layout by adding /layouts/template.php file, or by putting a new layout file right here THEME_NAME/styles/fresh/layouts/template.php. Now you can start customizing it.

A style folder can contain:

  • THEME_NAME/styles/fresh/css/ (CSS files)
  • THEME_NAME/styles/fresh/layouts/ (Template files, HTML, PHP)
  • THEME_NAME/styles/fresh/js/ (JavaScript files)
  • THEME_NAME/styles/fresh/images/ (Image files)

Adding the new style to the config.xml

To make your new style available in the theme settings you need to edit the THEME_NAME/config.xml Go to line ~29 and add a new style option:

<field name="style" type="list" default="" label="Style" description="Select the style variation.">
		<option value="">Default</option>
		<option value="header">Header</option>
		<option value="pop">Pop</option>
		<option value="blocks">Blocks</option>
		<option value="page">Page</option>
		<option value="fresh">Fresh (my new style)</option>
</field>

Important is that the value is the same like your styles folder name fresh. You can write whatever you want as description between the <option>....</option> tag.
