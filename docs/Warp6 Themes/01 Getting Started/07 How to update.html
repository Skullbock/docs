How to update

Here is a brief instruction on how to update a theme based on the Warp framework without uninstalling and reinstalling the template.

Update your theme

When you open the template settings a notice notify you whenever a new version of your template is available.Notifications are smart and won't bother you with any messages unless an update is available for your template.

Usually after you download a template it will be customized to meet the requirements of your website. In this process you will change images, add some HTML or do CSS modifications here and there.

At the bottom of the template settings the new file verification quickly shows you all the modified and missing files in your template that differ from the version you got from our download section. Now you instantly know which files you have modified in your customization before you start updating your individual template with the new template version.

To update your template do the following steps:

  1. First of all make a backup of all the customized template files, so you can recover it and redo all changes at any time.
  2. Download the latest template version and extract the files.
  3. Copy all files to templates/TEMPLATENAME/ (Joomla) or themes/THEMENAME (Wordpress) and replace all files.
  4. Redo your changes by merging your customized files with the new ones

To never have trouble with updating your theme we recommend that you write all your custom CSS styles into the file custom.css. That is the purpose of the custom.css file. Also check out the tutorial on how to make your template access custom CSS and JavaScript files.
