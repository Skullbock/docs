Install a Wordpress theme

In this tutorial we'll show you, how to install a template in your Wordpress installation.

To install a template based on the Warp framework use the template installer in the Wordpress administration. Go to Appearance » Themes and click on the tab Install Themes. Choose upload.

Click on Browse..., select the downloaded template file and click in Install Now.

Then go to Manage Themes and click on Activate to activate the newly installed template.

After the activation you will see a new item with a small YOO icon and the name of the template in the administration menu on the left. In the according subitems Theme Options and Widget Options you can change all template specific settings.

When the installation does not work this way you may try to install the template this way:
  1. Download the latest template version and unpack the templates files and folders in the archive in a temporary directory on your local hard drive.
  2. Using FTP, upload the template directory (including files and subdirectories) to the directory \wp-content\themes\ on your server.
  3. Go to Appearance » Themes in the Wordpress administration and the new theme will appear in the overview of installed themes.
  4. Click on Activate to activate the newly installed template.
